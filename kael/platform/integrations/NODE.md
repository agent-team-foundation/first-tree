# Integrations

Integrations connect Kael to external systems while keeping the core agent loop intact. They usually introduce account-level credentials, platform adapters, or execution backends rather than new product roots.

## Nodes

- [im-channels.md](im-channels.md) - third-party chat channels such as Feishu that mirror a Kael session outside the web UI.
- [coding-agent-integrations.md](coding-agent-integrations.md) - the current Claude Code delegation path and the extension boundary for future coding-agent providers.
- [kael-api.md](kael-api.md) - model tools (TTS, OCR, image generation, STT) exposed as a CLI package and HTTP endpoint, callable by any agent.
