# Channels

Chat is accessible through multiple channels, all converging on the same backend:

- **Web** — primary channel, full-featured SSE chat
- **Feishu** — bot integration, supports multi-bot per workspace
- **Desktop App** — Tauri system tray, same SSE protocol
- **CLI** — command-line interface
