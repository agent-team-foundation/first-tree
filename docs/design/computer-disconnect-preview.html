Decisions reflected in this iteration
  • Pulsing dot — exact same DOM pattern as StateDot.working: solid circle + absolute ring with animation: ring-pulse 1.8s infinite. Only the colour token changes (--state-error).
  • No icon — chip is dot + text only.
  • Copy — single: "<host> disconnected"; multi: "<n> computers disconnected".
  • Hostname max-width — 160px, text-overflow: ellipsis; full hostname surfaces in the chip's title tooltip.
  • Tab centring — topbar grid changed from auto 1fr auto to 1fr auto 1fr. Tabs always sit at the page centre regardless of how the brand cluster grows. The dashed green guide line in each frame proves this.
  • Waiting / success colours — yellow --state-blocked (warn) for waiting, green --state-idle (ok) for success. Both via color-mix so they sit on the same soft-pastel scale as other inline states.
