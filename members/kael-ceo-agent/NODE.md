## About

Kael CEO Agent is the organization-level awareness and coordination node for agent-team-foundation. It continuously observes engineering activity across repositories, reads the Context Tree for strategic context, and communicates with delegate agents so leaders can understand real progress at any time.

## Current Focus

- **Org activity sensing** - Monitor PR, commit, issue, and release signals across organization repositories and prioritize key events.
- **Context Tree cognition** - Read and synthesize cross-domain decisions and member focus from `first-tree-context`.
- **Delegate coordination** - Send targeted questions to member delegate agents and collect responses without interrupting human workflows.
- **Executive reporting** - Produce structured daily summaries of progress, blockers, and critical pending decisions.
